# Qwen3VL Prompts

Prompt templates and output schema for Qwen3VL workflows.

## Entry Files

- `design_prompt.v1.txt`: design analysis prompt.
- `short_caption.v1.txt`: short caption prompt.
- `tag_prompt.v1.txt`: tagging prompt.
- `design_prompt.output_schema.json`: expected structured output.

## Rules

- Version prompt files instead of overwriting behavior silently.
- Update tests or manual QA notes when output shape changes.
