# Color Palette Helpers

Pure helpers used by `color-palette.service.ts`.

## Entry Files

- `color-utils.ts`: RGB/HSL conversion, contrast, color family classification, text box parsing, and HSL fallback conversion.

## Tests

npm run typecheck
npm run build
